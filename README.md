**PROMPTS USED**:



&#x20;  Write a UGC-style Instagram ad script for Dot \& Key Vitamin C Serum.



Use an emotional and authentic tone.



Follow:

Hook → Problem → Solution → Results → CTA



Length: 30 seconds

Target audience: College students and young women.

1. My skin looked so dull until I found this serum!
2. If you're struggling with acne marks, watch this.
3. This skincare product changed my daily routine.
4. Everyone keeps asking what I'm using on my skin!
5. I wish I had started using this sooner.
